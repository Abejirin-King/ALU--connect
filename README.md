# ALU Connect

A Flutter-based platform connecting ALU students with internship, volunteer, and startup opportunities.

## Features

- **Role-Based Access**: Student and Startup accounts
- **Opportunity Discovery**: Browse and search opportunities
- **Application Tracking**: Applied, Review, Accepted, Rejected
- **Startup Dashboard**: Post opportunities and manage applicants
- **Firebase Backend**: Auth, Firestore, real-time updates
- **Cross-Platform**: Web, Android, iOS

## Tech Stack

- Flutter + Riverpod
- Firebase Auth & Firestore
- Material Design 3

## Setup

1. Clone the repo
2. Run `flutter pub get`
3. Configure Firebase (`flutterfire configure`)
4. Run `flutter run`

## Roles

- **Student**: Browse opportunities, apply
- **Startup**: Post opportunities, manage applicants
